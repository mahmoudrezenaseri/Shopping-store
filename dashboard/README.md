## Getting Started

Follow the steps below to run this project.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mahmoudrezenaseri/digikala.git
   ```
2. Install NPM packages

   Redirect to the Dashboard folder and run this command.

   ```sh
   npm install
   ```

3. Run the project
   ```sh
   npm start
   ```
